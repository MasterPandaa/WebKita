Web development

Terdapat beberapa bahasa pemrograman dan framework yang digunakan dalam pengembangan website, diantaranya:

1

HTML

Html adalah singkatan dari Hypertext Markup Language. Html merupakan bahasa pemrograman standar yang digunakan untuk membuat dan menampilkan halaman web.

2

css

CSS singkatan dari Cascading Style Sheets, adalah bahasa style sheet yang digunakan untuk mengatur tampilan elemen yang tertulis dalam bahasa markup seperti HTML.

3

Bootstrap

Bootstrap adalah sebuah framework CSS yang open source dan populer digunakan untuk mengembangkan front-end website maupun aplikasi web dengan cepat dan mudah.

4

PHP

PHP (Hypertext Preprocessor) adalah bahasa pemrograman server-side open source untuk menghasilkan dinamis web content dengan interaksi database. Dengan PHP, web developer dapat menghasilkan web dynamic, database-driven, dan e-commerce websites.

5

Javascript

JavaScript adalah bahasa pemrograman berbasis skrip yang digunakan untuk menambahkan interaktivitas dan dinamisme ke halaman web. Digunakan secara luas untuk mengendalikan perilaku halaman web, membuat animasi, dan menangani interaksi pengguna.

6

SQL

SQL (Structured Query Language) adalah bahasa pemrograman untuk menyimpan dan memproses informasi dalam basis data relasional. Sebuah basis data relasional menyimpan informasi dalam bentuk tabel, dengan baris dan kolom yang mewakili atribut data yang berbeda.
